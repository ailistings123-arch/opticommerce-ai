## 📊 Analytics Features

### Key Metrics Cards (4 Cards)

1. **Average Optimizations per User**
   - Shows how engaged users are
   - Calculated: Total optimizations ÷ Total users
   - Icon: Target

2. **Active Users**
   - Users who have created at least 1 optimization
   - Shows percentage of total users
   - Icon: Users

3. **Total Activity**
   - Total number of optimizations across all users
   - Icon: Activity

4. **Platforms Used**
   - Number of different platforms being used
   - Icon: Zap

---

## 📈 Charts & Visualizations

### 1. User Tier Distribution (Pie Chart)
- **What it shows**: Breakdown of users by subscription tier
- **Tiers**: Free, Starter, Professional, Enterprise
- **Format**: Percentage labels on each slice
- **Colors**: 
  - Free: Gray
  - Starter: Green
  - Professional: Blue
  - Enterprise: Purple

### 2. Platform Usage (Bar Chart)
- **What it shows**: Which platforms are most popular
- **Platforms**: Amazon, eBay, Etsy, Shopify, Walmart, WooCommerce
- **Sorted**: Highest to lowest usage
- **Format**: Vertical bars with counts

### 3. Top 10 Active Users (Horizontal Bar Chart)
- **What it shows**: Most active users by optimization count
- **Display**: Username (email prefix)
- **Sorted**: Most active first
- **Limit**: Top 10 users
- **Color**: Green bars

### 4. Usage by Tier (Composed Chart)
- **What it shows**: Optimizations and average per user by tier
- **Metrics**:
  - Blue bars: Total optimizations per tier
  - Green line: Average optimizations per user
- **Insight**: Shows which tiers are most engaged

### 5. Activity & User Growth (30 Days)
- **What it shows**: Daily optimizations and new user signups
- **Time range**: Last 30 days
- **Metrics**:
  - Blue area: Daily optimizations
  - Green line: New users per day
- **Format**: Dual-axis chart

### 6. Cumulative User Growth (Area Chart)
- **What it shows**: Total users over time
- **Time range**: Last 30 days
- **Format**: Purple gradient area
- **Insight**: Shows growth trajectory

### 7. Optimization Modes (Pie Chart)
- **What it shows**: Distribution of optimization types
- **Modes**:
  - Optimize Existing
  - Create New
  - Analyze URL
- **Format**: Percentage labels

### 8. Hourly Activity Pattern (Area Chart)
- **What it shows**: When users are most active (24-hour cycle)
- **Format**: Orange gradient area
- **Insight**: Peak usage hours
- **Use case**: Schedule maintenance during low-activity hours

### 9. Platform Performance Comparison (Radar Chart)
- **What it shows**: Relative usage across all platforms
- **Format**: 6-sided radar chart
- **Color**: Blue with transparency
- **Insight**: Visual comparison of platform popularity
